Comma-separated list of Git VCS repository patterns for which to enable Gradle Enterprise Gradle plugin/Maven extension auto-injection. By default, all Git VCS repositories have auto-injection enabled. If specified, the auto-injection will only apply to the projects with VCS URLs containing one of the said patterns Disabled nodes have precedence over this filtering. This feature is currently in Beta and supports Gradle Pipeline/FreeStyle and Maven Pipeline job configurations. Functionality can be changed in future releases.
